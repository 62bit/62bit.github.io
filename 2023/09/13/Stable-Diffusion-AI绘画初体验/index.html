Stable Diffusion AI绘画初体验

发表于2023-09-13|更新于2023-09-13
|阅读量:

AI绘画最近挺火的，我也来试试。一开始使用的是在自己电脑上本地部署，但是磁盘空间不够多，且显存也不够大，用起来很不爽。后来采用了应用市场 – 揽睿星舟-一站式AI服务平台和AI应用生态的Stable Diffusion Web UI打包应用，真的省了很多心。

以下是我在没有了解任何相关知识的情况下，用单一的提示词魔改了我大学社团的团徽：

Prompt 提示词

基础

  1. 提示词之间要用英文逗号分隔
  2. 提示词之间是可以换行的
  3. 权重默认为1，提示词越靠前权重越高
  4. 数量控制在75个单词以内

分类

  1. 内容提示词
  2. 画质提示词
  3. 画风提示词

建议顺序：画质 -> 画风 -> 内容 -> 环境 -> 画幅 -> Lora -> 负面词

修改提示词权重

  1. 圆括号(prompt)：权重变为1.1倍，最多套3层，变为1.331倍
  2. 大括号{prompt}：权重变为1.05倍，最多套3层，变为1.15倍
  3. 方括号[prompt]：权重变为0.9倍，最多套3层，变为0.729倍
  4. 数字权重(prompt:1.5)：数字代表权重变为原来的多少倍，建议在0.3~1.5之间
  5. 尖括号<lora:文件触发:权重>：调用lora
  6. 下划线prompt_prompt：让词与词之间更紧密地联系在一起

进阶语法

  1. 方括号加单冒号[flowers:0.7]：采样进程跑到70%后，对应提示词才开始生效
  2. 方括号加双冒号[flowers::0.7]：前70%的采样进程中，提示词生效
  3. 方括号双提示词加冒号[white flower:bush:0.8]：进程达到80%之前生成白花，80%之后生成灌木
  4. 交替采样[white|red|blue] flower：每采样一步换一个提示词
  5. 混合 white|yellow flower：生成黄色和白色混合的花

一些术语

采样步数：从模糊到清晰过程中绘制的次数

采样方法：生成图像时采用的算法
